# 프로젝트 요약
한국교통대학교의 커뮤니티 사이트를 제작 및 운영하였습니다.
현재는 주요 운영진들이 졸업을 하며 서비스 종료를 하게 되었습니다.

# 프로젝트 진행과정
- 'xpress engine' 프레임워크를 이용해 게시판 웹사이트를 제작한다.
- 'mysql'을 이용하여 DB를 설계한다.
- '카페24'를 이용하여 호스팅 및 도메인을 구매한다.
- '안드로이드 스튜디오' 툴을 사용하여 안드로이드 웹뷰 앱을 제작한다.
- 'swift'를 사용하여 ios 웹뷰 앱을 제작한다.

# 기술 스택
php / xpress engine / html / css / mysql / swift / java / android studio

# 팀원
*양동오
-> 총괄 *이혜은
-> 마케팅 *어동선
-> 디자인 및 웹디자인 *주한봉
-> 기술총괄(DB설계 / 로직설계 / 코드작성 / 디버깅 / 어플제작)

# 후기
이 프로젝트에서 처음으로 몇년에 걸친 장기간 서비스를 경험 해보게 되었습니다.
서비스를 진행하는 내내 끊임없는 유지보수 요청과 기능 고도화 요청을 받으며 왜 IT 서비스를 유지하는 것이 달리는 마차의 바퀴를 갈아끼는 것이라고 비유되는지 실감하게 되었습니다.
카페24를 통해 호스팅을 하고 도메인을 구매했었는데, 커다란 이벤트를 진행 할 때 트래픽 초과로 서버가 마비되는 것도 처음으로 경험하게 되었습니다.
프로젝트를 진행하며 기술적인 향상도 향상이지만 특히나 문제에 대처하는 능력이 크게 올라간 것 같습니다.
