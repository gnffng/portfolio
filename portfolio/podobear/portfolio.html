# 프로젝트 요약
명함을 촬영하면 명함의 정보들을 자동으로 뽑아내어 연락처로 가공해주는 애플리케이션
서버를 따로 두지 않고 통신 없이 앱 자체적으로 AI 모듈을 사용함

# 프로젝트 진행과정
- React Native를 기반으로 프로젝트를 진행한다.
- AsyncStorage를 사용하여 디바이스 내부 저장공간을 DB로 사용한다.
- 각종 기능과 UI를 구현한다.
- 영역인식, 텍스트 인식 등의 AI 모듈을 React Native에 이식한다.
- 인식률 최적화 및 용량 최적화를 시행함.
- PlayStore에 절차를 따라 앱을 런칭함.

# 기술 스택
reactNative

# 팀원
*주한봉
-> DB설계 / AI 모듈을 제외한 모든 기능 구현 / UI 화면 구현 / 테스트
*김난희, 박소정
-> AI 모듈 / 테스트

# 후기
회사의 내부 프로젝트로 진행된 프로젝트입니다.
원래는 제가 속한 팀의 프로젝트가 아닌 다른 팀의 프로젝트였지만, 해당 팀에 앱개발이 가능한 개발자가 없었기 때문에 제가 앱개발로 파견을 가게 되었습니다.
하이브리드 플랫폼 프레임워크를 처음 접해보기도 헀고, React의 숙련도가 낮은 상태로 진행했던 프로젝트였기 때문에 맨땅에 헤딩을 하는 기분이었습니다.
회사에 앱 개발자가 단 한명도 없었기 때문에 하나하나 스스로 시행착오를 겪어야했고, 노베이스의 기술스택으로 멘토도 없었기 때문에 개발의 시작부터 끝까지 끊임없는 리펙토링의 연속이었습니다.
그래도 끝까지 프로젝트를 마치고 런칭까지 마무리 했기 때문에 후련했고, 도전하기 좋은 기회였던 것 같습니다.
