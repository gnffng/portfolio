# 프로젝트 요약
졸업작품으로 프로젝트를 진행하게 되었습니다.
일반적인 스마트팜과는 다르게 생장속도와 출하속도를 단계별로 제어할 수 있는 시스템을 만들었습니다.
설정값에 따라 일조량, 급수량을 제어하는 시스템입니다.
# 프로젝트 진행과정
- 시스템에 필요한 기능들을 정리하고 정리된 기능들을 모듈화하며 로직을 설계한다.
- 각 기능들에 필요한 부품들을 확보한다.
- 'Raspberry Pi'를 이용해 모든 연산을 처리하는 서버를 만든다.
- 'Arduino'를 이용해 입출력 제어 시스템을 만든다.
- 'Raspberry Pi'와 'Arduino'가 통신할 수 있는 커스텀 프로토콜을 만들어 시리얼통신을 한다.
- 3D 모델링으로 프로젝트에 필요한 외골격을 제작한다. - 어댑터와 '7805C' 소자를 이용하여 220V 전압을 12V, 5V로 변환하고 각 부품들에 공급한다.

# 기술 스택
python / c / raspberry pi / arduino / serial / solidworks / protocol

# 후기
통신 로직을 개발할 때 아두이노가 제공하는 시리얼 단자를 필수적으로 사용하는 부품이 있어서 시리얼 통신을 할 핀이 부족하다는 이슈가 발생했었습니다.
여러 자료를 참고한 끝에 'SoftwareSerial'이라는 모듈을 통해 인위적으로 입출력 핀을 시리얼 통신 핀으로 변환을 해서 문제를 해결할 수 있었습니다.
그리고 로직을 처리하는 서버와 입출력을 제어하는 클라이언트를 분리하여 개발하는 것이 쉽지 않았습니다.
제어부에서 연산까지 감당했다면 서버도 통신도 필요하지 않았을 것이지만, 다른 두 기기의 통신 모듈을 각각 개발하는 것은 손이 많이 가는 일이었습니다.
처음부터 끝까지 많은 이슈가 있었고 손도 많이 가고 신경쓸 것도 많은 프로젝트였습니다.
심지어 저는 계절학기로 졸업작품을 만들게 되어 개발기간이 3주밖에 주어지지 않아서 정말 정신없이 프로젝트를 진행하게 되었습니다.
쉽지 않은 경험이었지만, 정해진 기간에 어떻게든 결과물을 완성해내자 뿌듯함과 큰 자부심을 느낄 수 있었습니다.
