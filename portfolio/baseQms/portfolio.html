# 프로젝트 요약
링크사업단의 일환으로 기업 선앤두의 홈페이지를 제작하게 되었습니다.

# 프로젝트 진행과정
- 키오스크, 주문어플의 주문을 받는 서버프로그램을 설계한다.
- 클라이언트의 주문 CRUD를 처리하는 RestAPI를 제작한다.
- 관리자 페이지를 제작하여 주문 및 상품 등을 사이트에서 제어할 수 있도록 한다.
- 주문이 들어오면 관리자 페이지에 실시간 알람이 울리도록 한다.
- 'spring boot' 프레임워크를 이용해 웹사이트를 제작한다.
- 'thymeleaf', 'bootstrap' 라이브러리를 사용하여 관리자 페이지 뷰를 구성한다.
- 'mysql'을 이용하여 DB를 설계한다.
- 레이어드 아키텍쳐를 사용하여 DB를 설계한다(View, Controller, Service, Repository, DAO).

# 기술 스택
spring boot / mysql / thymeleaf / bootstrap / spring scurity / restAPI / postman

# 팀원
*주한봉(개인프로젝트)
-> DB설계 / 로직설계 / 코드작성 / 테스트 / 디버깅

# 후기
'spring boot' 프레임워크를 공부하며 진행한 프로젝트입니다.
작은 RestAPI에서 시작하여 관리자 페이지, 보안, 실시간 소켓통신 등 하나씩 배운 기능들을 더해가며 풍부해져가는 프로젝트를 보니 뿌듯한 마음이 들었습니다.
처음에는 공부를 할 겸 가벼운 마음으로 시작한 프로젝트였지만, 내가 소비자라면 어떤 기능들이 필요할까를 생각하며 기능을 추가해 나가다 보니 1년이 넘게 프로젝트를 진행하게 되었습니다.
모든 개발은 산출물을 소비하는 클라이언트의 시선에 맞추어 진행해야 한다는 것을 느끼게 되었습니다.
나중에 여유가 되면 클라이언트 어플을 제작하여 통신을 해볼 것 같습니다.
