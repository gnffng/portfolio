# 프로젝트 요약
시청에서 사무보조를 하다 많은 양의 지번주소를 도로명주소로 변환하는 작업을 하게 되었습니다.
네이버에 일일히 지번주소를 검색하여 도로명주소를 알아내고, 알아낸 도로명주소를 엑셀파일에 옮기는 단순한 반복작업이었습니다.
며칠 정도는 걸릴 분량의 업무량이었지만, 저는 웹크롤링 프로그램을 만들어 하루만에 작업을 끝낼 수 있었습니다.

# 프로젝트 진행과정
- 'pywin32' 라이브러리를 이용해 엑셀 파일의 지번주소를 파이썬 프로그램에 읽어온다.
- 크롬 브라우저를 제어할 수 있게 해주는 'Selenium' 라이브러리를 이용하여 지번주소에 해당하는 도로명주소를 검색한다.
- 읽어온 도로명주소를 'pywin32' 라이브러리를 통해 엑셀 파일에 업데이트한다.

# 기술 스택
python / Selenium / pywin32
