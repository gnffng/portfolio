# 프로젝트 요약
DB설계, MVC패턴, CRUD 연습을 목적으로 제작한 카피 사이트입니다.
(원본사이트 : https://rollingpaper.site/)

# 프로젝트 진행과정
- 'dJango' 프레임워크를 이용해 MVC패턴 웹프로그램을 제작한다.
- 'dJango'의 ORM을 이용하여 Board(게시판), Post(게시글) 등 DB를 설계한다.
- 사용자가 각각 고유한 ID값으로 게시판을 생성할 수 있게 한다.
- 다른 사용자들이 생성된 게시판에 게시글을 올릴 수 있도록 한다.
- 게시판과 게시글에 관한 CRUD를 제공한다.

# 기술 스택
python / dJango / hashlib

# 팀원

# 후기
특별한 로직 없이 기본적인 기능만 하는 프로그램을 만드는 것도 생각보다 손이 많이 간다는 것을 체감했습니다.
특히 프론트엔드 영역이 복잡한 고민이나 설계가 필요하지는 않지만 정직하게 시간이 많이 들어가는것 같습니다.
