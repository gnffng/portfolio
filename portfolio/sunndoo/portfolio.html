# 프로젝트 요약
링크사업단의 일환으로 기업 선앤두의 홈페이지를 제작하게 되었습니다.

# 프로젝트 진행과정
- 기존 사이트에 있던 자료들을 정리하고 기업 임원진과 상의를 통해 사이트를 구성할 내용들을 선별한다.
- 디자인 구상, DB 설계, 리소스 확보 등 사전 작업을 한다.
- 'dJango' 프레임워크를 이용해 웹사이트를 제작한다.
- 'mysql'을 이용하여 DB를 설계한다.

# 기술 스택
python / dJango / html / css

    # 후기
    처음으로 기업에서 개발업무를 수행하게 되었습니다.
    IT기업이 아닌 제조업 회사로 현장실습을 가게 되어 현장실습을 간 학생들끼리 주도적으로 개발을 진행하는 귀한 경험을 해볼 수 있었습니다.
    개인 프로젝트나 학생들끼리의 팀프로젝트를 할 때는 대체로 내가 선택하고 내가 결정하여 프로젝트를 진행했었지만, 기업에서 받은 업무는 다른 사람이 바라는 니즈를 충족시키는 프로젝트라는 점이 큰 차이점이었던 것 같습니다.
    개발자에게는 실질적인 개발역량인 하드스킬도 중요하지만 소통을 위한 소프트스킬도 정말 중요하다는 것을 느꼈습니다.
    3개월이라는 짧은 시간이었지만 많은 것을 배우고 성장할 수 있던 시간이었다고 생각합니다.
