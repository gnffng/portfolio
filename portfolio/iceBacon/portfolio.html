# 프로젝트 요약
삼성전자의 LSI 결함 검출 시스템의 개발.

# 프로젝트 진행과정
- API는 FastAPI를 사용하여 개발함.
- UI는 React를 사용하여 개발함.
- DB는 MongoDB를 사용하고 GraphQL로 제어함.
- 삼성전자의 LSI의 정보를 받아서 DB에 셋팅함.
- 웨이퍼의 결함 데이터를 json으로 가공하여 입력함.
- 화면에서 각종 option을 통해 결함정보를 스케일에 맞게 출력함.

# 기술 스택
fastApi / mongoDB / graphQL(ariadne)
