삼성 LSI 결함 검출 시스템 개발

Description

  • 주한봉 외 3명

  • 2023.01~2023.03


# 프로젝트 요약
삼성전자의 LSI 결함 검출 시스템의 개발.

# 프로젝트 진행과정
- API는 FastAPI를 사용하여 개발함.
- UI는 React를 사용하여 개발함.
- DB는 MongoDB를 사용하고 GraphQL로 제어함.
- 삼성전자의 LSI의 정보를 받아서 DB에 셋팅함.
- 웨이퍼의 결함 데이터를 json으로 가공하여 입력함.
- 화면에서 각종 option을 통해 결함정보를 스케일에 맞게 출력함.

# 기술 스택
fastApi / mongoDB / graphQL(ariadne)

# 후기
개발 프로젝트의 끝자락에 백업 인원으로 합류하여 페이징 처리를 담당하였습니다.
처음으로 접해보는 GraphQL이여서 DB를 다루는 새로운 관점이 신선했습니다.
대기업의 개발 프로젝트를 경험해보고 분위기를 느껴볼 수 있었던 좋은 경험이었던 것 같습니다.
